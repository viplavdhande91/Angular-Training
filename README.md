
# Material UI

## API Reference

#### Get all items

#### 1. Import module in app.module.ts For Example
```
   import { MatSliderModule } from '@angular/material/slider';

```

#### 2. Register module in imports of app.module.ts For Example
```
imports: [
    BrowserModule,
    AppRoutingModule,
    BrowserAnimationsModule,
    MatSliderModule,
    NgbModule,
    CdkAccordionModule
  ]
```
